# 6. 파일 시스템



### 1. 파일 시스템

* 파일
  * 데이터의 논리적 집합
* 모든 데이터, 프로그램의 저장 & 접근을 쉽게하는 OS의 한 부분
* 특징
  * 커널 영역에서 동작
  * 파일 CRUD 를 원활히 수행하기 위함
  * 계층적 디렉터리 구조
  * 디스크 파이션 별로 하나씩 둘 수 있음
* 역할
  * 파일 / 보조 기억장치 관리
  * 파일 무결성 메커니즘
  * 접근 방법 제공
* 목적
  * 메인메모리와 하드디스크 간 속도 차 줄이기
  * 파일 관리
  * 하드디스크 용량 효율적 이용
* 구조
  * 메타 영역
    * 데이터 영역에 기록된, 파일 이름 / 위치 / 크기 / 시간 정보 / 삭제 유무 등 파일 정보
  * 데이터 영역
    * 파일의 데이터
